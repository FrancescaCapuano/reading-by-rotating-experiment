# reading-by-rotating (custom jsPsych plugin)

## Overview
This is a custom plugin developed for a behavioral experiment implementing a *reading-by-rotating* paradigm.

The plugin enables participants to control stimulus presentation via rotational input, simulating a physical knob interaction in an online environment.

## Context
The plugin was designed to adapt a classic experimental paradigm (Action-Sentence Compatibility Effect, ACE) to a browser-based setting.

It is integrated into the main experiment pipeline (`experiment.js`).
